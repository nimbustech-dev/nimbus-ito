# 🔄 수정한 코드를 서버에 반영하는 가이드

로컬 컴퓨터(윈도우)에서 수정한 코드를 깃허브(GitHub)를 통해 NCP 서버에 아주 간단하게 반영하는 과정입니다.

---

### 1단계: 내 컴퓨터(로컬)에서 수정 및 업로드

먼저 내 컴퓨터(`C:\nimbus-ito`)에서 코드를 수정하고 깃허브에 올리는 과정입니다.

1. **터미널 실행**: `Win + R` 키를 누르고 `powershell`을 입력해 새 창을 엽니다.
2. **프로젝트 폴더로 이동**:
   ```powershell
   cd C:\nimbus-ito
   ```
3. **수정된 코드를 깃허브에 올리기**:
   ```powershell
   git add .
   git commit -m "대시보드 로직 수정"
   git push origin main
   ```

---

### 2단계: NCP 서버에 접속하기

이제 서버가 최신 코드를 내려받도록 명령을 내려야 합니다.

1. **서버 접속**: (인증키가 있는 폴더에서 실행. 예: `cd C:\Users\lenovo\Desktop`)
   ```powershell
   ssh -i nimso2026.pem root@211.188.57.42
   ```
2. **비밀번호 입력**: 서버 비밀번호(`A6+bYq5br7Tt$h`)를 입력합니다. (화면에 글자가 안 보여도 입력 후 엔터를 치시면 됩니다.)

--- 

### 3단계: 서버에서 최신 코드 불러오기 및 재시작

이미 `git clone`을 해두었기 때문에 `pull` 한 줄이면 업데이트가 끝납니다.

1. **프로젝트 폴더로 이동**:
   ```bash
   cd /root/deal-ito
   ```
2. **최신 코드 내려받기**:
   ```bash
   git pull origin main
   ```
   ⚠️ **에러 발생 시** (`dashboard_cache.pkl` 충돌): 서버에 저장된 엑셀 캐시 때문에 pull이 막힐 수 있습니다. 아래를 먼저 실행한 뒤 다시 pull 하세요.
   ```bash
   git checkout -- dashboard_cache.pkl
   git pull origin main
   ```
3. **프로그램 재시작 (권장)**:
   Streamlit은 파일 변경을 감지하지만, 확실하게 반영하기 위해 프로세스를 껐다 켭니다.
   ```bash
   # 1. 기존 실행 중인 대시보드 종료
   pkill -f streamlit

   # 2. 백그라운드 재실행
   nohup streamlit run app.py --server.port 8501 &
   ```

---

### 💡 팁: 이 과정을 더 줄이고 싶다면? (업데이트 전용 명령어 만들기)

서버 터미널에서 아래 명령어를 한 번만 입력해두면, 나중에는 **`update`**라는 글자만 쳐도 위 과정이 자동으로 실행됩니다.

```bash
alias update='cd /root/deal-ito && git pull origin main && pkill -f streamlit && nohup streamlit run app.py --server.port 8501 &'
```

* 위 설정을 해두면 나중에는 서버 접속 후 **`update`** 한 단어만 치면 끝납니다!

---

### 🔄 서버 재시작 시

NCP 콘솔에서 서버를 **재시작**하면 Streamlit이 꺼집니다. 아래 순서로 다시 실행하세요.

1. **NCP 콘솔** → Server → **deal-ito** 선택 → **서버 접속 콘솔** 클릭
2. 로그인: `root` 입력 후 엔터 → 비밀번호(`A6+bYq5br7Tt$h`) 입력 후 엔터
3. Streamlit 다시 실행 (한 줄씩, 오타 주의: app.py 랑 -- 사이에 공백):
   ```bash
   cd /root/deal-ito
   nohup streamlit run app.py --server.port 8501 &
   ```

* SSH가 되면 `ssh -i nimso2026.pem root@211.188.57.42` 로 접속 후 위 3번 명령을 실행해도 됩니다.
* 인증키(PEM) 위치: `C:\Users\lenovo\Desktop\nimso2026.pem`
